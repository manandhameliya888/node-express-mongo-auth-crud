# NodeJS ExpressJS and MongoDB AUTH + CRUD REST API 

## Description
This is an Contact Manager App made with NodeJS ExpressJS and MongoDB AUTH + CRUD REST API with JWT and all Request Response validation

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
